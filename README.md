# SMS-Email-Marketing
Enterprise-grade ASP.NET Core application demonstrating clean architecture, high-performance APIs, background processing, and scalable system design using real-world patterns.

This repository showcases a production-style ASP.NET Core application designed with scalability, performance, and maintainability in mind. 
It follows clean architecture principles, uses repository & service patterns, and demonstrates how to build high-throughput APIs suitable for real enterprise systems.
